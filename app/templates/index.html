Humanitarian Data Space

Find, explore, and request access to humanitarian datasets — all in one place

What does this platform do?

The Humanitarian Data Space helps researchers and humanitarian organizations discover datasets that are relevant to their work. Many valuable datasets exist across different data repositories, but finding and accessing them can be difficult. This platform brings them together so you can:

  • Search across multiple data sources — connect to data repositories and see all available datasets in one overview
  • Filter and explore — narrow down datasets by topic, keyword, or data provider
  • Request access — select the datasets you need and generate a ready-to-send request email to the data owner
  • Query data directly — for datasets with query endpoints, run analyses without downloading the data
