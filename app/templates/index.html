What is Data Visiting?

Data visiting (code-to-data) is an approach where your analysis queries are sent to datasets for local execution, returning only verified results. This tool helps you:

  • Discover datasets across multiple FAIR Data Points
  • Browse and filter datasets by theme or keyword
  • Compose standardized data access request emails
  • Send requests to dataset contact points
